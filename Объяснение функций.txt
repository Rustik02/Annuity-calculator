functions.js: Этот файл содержит функцию calculateAmortizationSchedule, которая принимает параметры суммы кредита, годовой процентной ставки и срока кредита в месяцах, а затем рассчитывает график аннуитетных платежей. Функция возвращает массив объектов, представляющих расписание платежей.

app.js: Этот файл содержит основной JavaScript-код для обработки событий на странице. Он устанавливает обработчик событий на форму ввода данных о кредите, который вызывает функцию calculateAmortizationSchedule при отправке формы. Затем он вызывает функцию displayAmortizationSchedule, чтобы отобразить полученное расписание платежей на странице.
